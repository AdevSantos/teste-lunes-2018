O Bitcoin é uma rede de consenso que permite um novo sistema de pagamento e um dinheiro totalmente digital.

Litecoin é uma moeda de Internet peer-to-peer que permite pagamentos instantâneos, com custo quase zero, para qualquer pessoa no mundo.
